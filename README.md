# CodesToDiagrams

### install pip
sudo apt install python3-pip

### update
sudo apt-get update

### install graphviz
sudo apt install graphviz

### using pip (pip3)
$ pip install diagrams

### using pipenv
$ pipenv install diagrams

### using poetry
$ poetry add diagrams
